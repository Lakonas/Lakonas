### What I'm building

🛡️ **[Phylax](https://phylax-7jth.vercel.app)** — Incident management for small teams, built by someone who spent 17 years managing incidents at scale. SLA/P queue sorting, AI severity suggestions and postmortem generation via Anthropic Claude, Little's Law dashboard, enforced state machine, role-based access control, and 35 tests (Jest + Playwright). Next.js, TypeScript, PostgreSQL with raw SQL. **[Repo →](https://github.com/Lakonas/Phylax)**

📈 **[Analytics Service](https://analytics-service-production-37cd.up.railway.app)** — A service-oriented event analytics platform. Any app sends events via POST, the service stores them in PostgreSQL with a generic JSONB schema, and a server-side dashboard aggregates everything with dynamic filtering. PhotoLabs and ShortStop both send real events to it.

📸 **[PhotoLabs](https://photolabs-sand.vercel.app)** — Full-stack photo gallery with JWT auth, Cloudinary uploads, and AI-powered descriptions via Anthropic's Claude Vision API. React frontend on Vercel, Express backend on Railway.

🔗 **[ShortStop](https://tinyapp-production-12b4.up.railway.app)** — URL shortener with click tracking analytics, QR code generation, and JWT auth. EJS/Express on Railway.
